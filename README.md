# arp-scanner

# Requisitos
Debes instalar el modulo scapy de Python, para ello ejecuta el siguiente comando
```
pip install scapy
```

# Uso
Para utilizar el script ejecuta como root
```
arpscanner.py -i <interfaz> -r <rango>
```
